maximize-image

Maximize images easily using CSS3 transforms.

Usage

maximizeImage("img");

Options

var opts = {
  overlay: true,
  // Boolean - overlay toggle
  bg: "rgba(255, 255, 255, .65)"
  // String - CSS color value for overlay background
};

maximizeImage("img", opts);
